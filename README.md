## Уровень 1: Начальный (Базовый синтаксис)

### Задание 1.1: Простая блок-схема «Приготовление чая»

Ниже представлена блок-схема процесса приготовления чая, выполненная с помощью Mermaid.

```mermaid
graph TD
    A[Начало] --> B[Кипячение воды]
    B --> C{Есть ли чай?}
    C -- Да --> D[Заварить чай]
    C -- Нет --> E[Купить чай]
    E --> D
    D --> F[Пить чай]
    F --> G[Конец]
